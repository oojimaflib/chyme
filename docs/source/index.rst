Welcome to Chyme's documentation!
=================================

Chyme is an API for accessing and updating hydraulic models. It is built around
a generalised storage concept for storing model data and contains functionality
for reading and writing to different modelling formats.

Chyme currently supports to the following model types::
- Flood Modeller
- ESTRY-TUFLOW

There are plans to also include::
- HEC-RAS
- Infoworks ICM
